## 📌 Sobre o Projeto

O Kadem é um ecossistema de produtividade projetado para garantir foco e organização. Construído com uma arquitetura **Offline First**, o sistema opera de forma 100% funcional sem conexão com a internet, sincronizando os dados automaticamente assim que a rede é restabelecida. A segurança é tratada como prioridade, com criptografia de ponta a ponta (E2EE) implementada em toda a plataforma.

## ✨ Funcionalidades Principais

- **Kanban Interativo:** Gestão visual de projetos, delegação de tarefas e organização ágil do fluxo de trabalho.
- **Radio Flow:** Integração de músicas e playlists do YouTube diretamente na plataforma, com suporte offline para manter o estado de foco.
- **Sincronização Offline:** Uso ininterrupto, independente da estabilidade da rede.
- **Segurança E2EE:** Proteção total dos dados e tarefas gerenciadas na aplicação.
